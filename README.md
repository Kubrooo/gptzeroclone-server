# GPTZeroClone Server

This repository contains the backend server for a GPTZero clone application. It is a lightweight REST API built with Node.js and Express, designed to receive text input and return an analysis.

This server is intended to be the backend component that powers a front-end application, handling the logic for text processing and analysis.

## Features

* A simple REST API endpoint for text analysis
* Built with Node.js and Express for reliable performance
* Includes CORS middleware to allow requests from different origins (e.g., a front-end web app)
* Uses dotenv for easy environment variable management
* Lightweight and ready for containerization or deployment

## Technologies Used

* Node.js
* Express.js
* cors (Node.js CORS middleware)
* dotenv (Loads environment variables from .env file)
